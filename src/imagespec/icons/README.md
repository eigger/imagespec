# Bundled icons (Material Design Icons)

The `icon` element needs the MDI webfont and its codepoint metadata. Copy both
files here before enabling the `icon` handler:

- `materialdesignicons-webfont.ttf`
- `materialdesignicons-webfont_meta.json`

Source: https://github.com/Templarian/MaterialDesign-Webfont/tree/master/fonts
(both components already ship these — copy from either
`hass-gicisky/.../fonts/` or `hass-niimbot/.../`).

`RenderContext.icons_dir` defaults to this folder.
